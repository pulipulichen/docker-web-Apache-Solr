# docker-web-apache-solr

這是為了資訊儲存與檢索課程的教學，將Apache Solr包裝成容易架設與設定的形式。

此配置是建置了Linux Debian bullseye作業系統以及完整的Apache Solr 8.7，並以Docker容器化包裝。教學時不會操作底層，提供有開發者自行探索。
